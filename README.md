# Real-Time Collaborative Document Editor

A real-time collaborative document editor built with **React**, **Node.js**, **Express**, and **MongoDB**. This project allows multiple users to edit documents simultaneously.

## Features

- Real-time collaborative editing
- User authentication and role-based access
- Create, edit, and delete documents
- Live updates with WebSockets or Socket.IO
- Simple and clean UI with React

## Installation

1. **Clone the repository**:

```bash
git clone <your-repo-url>
cd <your-repo-folder>

2. **Install dependencies (both frontend and backend)**:

```bash
npm install

3. **Run the project (frontend & backend together)**:

```bash
npm run dev
